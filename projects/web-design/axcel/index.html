Project Overview

Axcel is a modern chat application designed to provide a seamless communication experience similar to Discord. The project focused on creating an intuitive landing page and a streamlined sign-up process that would attract and retain users while showcasing the app's key features and benefits.

My Role

I was responsible for the complete design and development of the user interface, including:

  • Landing Page Design - Creating an engaging homepage that highlights key features
  • Sign-up Flow Design - Designing a user-friendly registration process
  • UI/UX Design - Developing the overall user experience and interface design
  • Responsive Design - Ensuring the design works perfectly across all devices

Design Process

The design process involved several key phases:

  1. UI Design - Developing high-fidelity designs with modern aesthetics
  2. Prototyping - Building interactive prototypes to test responsiveness
  3. Development - Finialzing the look
