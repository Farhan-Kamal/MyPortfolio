Overview

Synthwave Runner is an endless runner game that captures the aesthetic and energy of 80s synthwave music. Players navigate through neon-lit environments while avoiding obstacles and collecting power-ups.

Key Features

  • Procedurally generated neon environments
  • Dynamic lighting and particle effects
  • Power-up system with visual feedback
  • Adaptive difficulty scaling

Technical Details

Built in Unity using C#, the game features object pooling for performance, procedural generation algorithms for endless gameplay, and a modular power-up system. The visual effects are achieved through Unity's particle system and post-processing stack.

Results & Impact

This project helped me master Unity's core systems and understand game design principles like difficulty curves and player engagement. The aesthetic focus taught me the importance of cohesive visual design in creating memorable experiences.
